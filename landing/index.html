Features

Built for creators,
not corporations.

Everything you need to run a real radio station from your own hardware — with none of the lock-in.

📻
Always Live

Your station streams 24/7 from your local vault via HLS. Music, beats, sessions, podcasts — FFmpeg handles the broadcast engine.

🗄️
Smart Archive

Point Paperweight at any folder. It scans with chokidar, probes with ffprobe, and keeps your library current automatically.

🔒
Self-Hosted

Runs entirely on your machine. No subscription, no cloud required. Share via Cloudflare Tunnel or your own domain.

💳
Monetization

Accept tips, sell subscriptions, and gate premium content. Stripe and PayPal wire payments directly to your accounts — Paperweight takes nothing.

🎫
Access Control

Issue listener tokens, define tiers (free, supporter, pro, all-access), and set per-track or per-project vault pricing.

📊
Creator Dashboard

Manage media, schedule broadcast blocks, monitor analytics, handle tokens, configure payments, and go live — all from a single local dashboard.

How It Works

Up in minutes.

No DevOps experience required. If you can run a terminal, you can run a station.

01
Install on any machine

Download the binary for your platform, or clone the repo and run with Node. Windows, macOS, Linux, and Raspberry Pi all supported.

npm start
02
Point at your folder

Drop your audio and video files in the vault directory. Paperweight scans automatically, extracts metadata, and builds your library.

vault/ → auto-scanned
03
Go live

Your station starts broadcasting immediately. Share your URL directly, or tunnel via Cloudflare — no port-forwarding headaches.

HLS stream ready
